The Frequency analysis scripts are used to investigate the following effects:
1. Pulse height as a function of the frequency used to generate them
    -> fs_frequency_analysis.m
    -> map_fs_comparison.m
    -> fit_exp_retention.m
2. The effect of downsampling on a single pulse and an entire rztp map
    -> downsampling_effect.m
3. The Hydrophone filter and its effect as parameterised by Cecile
    -> Hydrophone_filter_Cecile.m 
4. Reading and fitting to the hydrophone data and the directionality
    -> read_complex_tf_file.m
    -> Hydrophone_fit_from_data.m

The Transfer_functions folder contains data from frequency sweeps done on 
a double membrane laser hydrophone.
